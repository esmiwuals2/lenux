GParted

Click the button below to start GParted if it's not already running.

To wipe a disk and install only Lenux:

  1. In the top-right, click the disk selection button. It has an icon of a disk beside it.
  2. Select the disk that you wish to install Lenux on, and ensure that it is not the disk that contains this installer. If it is, the next step will result in error.
  3. Click Device > Create Partition Table. Choose MBR for the partition table type. Click Apply. BEWARE: THIS OPERATION WILL ERASE ALL DATA ON THE SELECTED DISK. BE VERY SURE YOU WANT TO DO THIS.
  4. Now jump to step 5 of "To install Lenux alongside another OS"

To install Lenux alongside another OS:

  1. In the top-right, click the disk selection button. It has an icon of a disk beside it.
  2. Select the disk that you wish to install Lenux on, and ensure that it is not the disk that contains this installer. If it is, the next step will result in error.
  3. Click View > Device Information if the info pane isn't shown. Ensure that the partition table is type MBR.
  4. If necessary, Select a partition and click Partition > Resize/Move. Shrink the partition by moving the ends, but ensure that it doesn't get smaller that the amount of data on it (shown in yellow). Click Resize/Move.
  5. Click Partition > New. All the settings here are correct, do not change them. Click Add.
  6. Click the green check mark at the top and click Apply to write the changes to the disk. When it says: "All operations successfully completed", Click Close.
  7. Note the value starting with /dev/ on the blue partition you created in step 5. You need it for the next few sections.
  8. Close GParted.
